# Tourism Admin Guide

## Overview
This administrative panel empowers Tourism Authorities to rapidly deploy, track, and manage physical checkpoints at Ethiopian Heritage Sites.

## 1. Landing / Analytics
Provides temporal scan statistics. Useful for understanding "when" peak tourism interacts with specific plaques. The Heatmap visually defines hotspots to target maintenance or extra features.

## 2. Managing Sites (Landmarks)
Navigate to **Sites**. Provide English and Amharic names, set points awarded, and precisely click the GPS location. Upon creation, a QR code specific to that location is produced automatically.

## 3. Physical Deployment (QR Bulk)
Next to "Add Site", click **Printable PDF** to download a print-ready A4 document displaying the QR codes and titles for cutting and physically mounting them.
